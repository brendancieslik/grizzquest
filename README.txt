GRIZZQUEST FINAL
Here's the final product of our game, GrizzQuest.
To run the game, simply download this game and launch the executable file!

If you would like to mod the game to your liking you can add more monsters and random events.
Download our Map branch and open the files using Unity.
If you would like to add your own custom graphics, use the Unity UI to add more game objects and adjust their colors and sizes.
To add more random events, navigate to "ASSETS/SCRIPTS/EVENTS.XML". Follow the template and have fun!
To add more monsters, navigate to "ASSETS/SCRIPTS/MONSTERS.XML". Follow the template and have fun!

This program was created by the Oakland University GrizzQuest team.
